Release Notes for The Debugging Book¶

This book comes with version numbers; these correspond to the version numbers in the Python pip package.

Version 1.x (in progress)¶

  • Nothing new at this point

Version 1.1 (2021-10-13)¶

  • We now require Python 3.9 or later.
  • Made an optimization in DeltaDebugger.dd() to distribute changes more evenly over the input space. This will change dd() results in some cases, but overall improve performance. Thanks to Olad Chitil and Joanna Sharrad for reporting this!
  • Fixed two issues in DeltaDebugger.dd() (#45, #46) Thanks to @TheSilvus for reporting this!
  • Added missing dependencies to the debuggingbook pip package] (#44)) Thanks to @TheSilvus for reporting this!
  • Fixed a warning '.gitignore is a symbolic link' during git checkout (#43)) Thanks to @rjc for reporting this!

Version 1.0.1 (2021-06-08)¶

  • Typos and other minor fixes.

Version 1.0 (2021-06-01)¶

  • First complete version.

Version 0.9.0 (2021-10-23)¶

  • Initial beta version.
